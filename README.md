# Ryan Basheer – Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, JavaScript, and Tailwind CSS. This portfolio showcases my education, technical skills, academic projects, and interests as a first-year BSc Computer Science student.

## 🌐 Live Demo
Add your deployed website link here:

https://ryan-basheer-portfolio.vercel.app/

---

## 📖 About

This portfolio was created to present my academic background, technical skills, and projects in a professional and visually appealing way. It also serves as a learning project to improve my frontend development skills.

---

## ✨ Features

- Responsive design for desktop and mobile devices
- Dark and blue modern UI theme
- Hero section with profile image
- About Me and Education sections
- Technical Skills showcase
- Academic Projects section
- Contact section with social links
- Floating social media island footer
- Smooth scrolling and subtle animations
- GitHub repository links for projects

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- Font Awesome Icons
- Google Fonts
